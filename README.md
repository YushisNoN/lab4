## Язык программирования
### Синтаксис
Язык представляет собой **ассемблер** с поддержкой:
- Меток (labels)
- Секций (`.text`, `.data`)
- Директивы `.org` для размещения кода по конкретным адресам
- Констант и литералов

#### Формат инструкций
| Формат | Синтаксис | Пример |
|--------|-----------|--------|
| R-тип | `OPCODE Rd, Rs1, Rs2` | `ADD R1, R2, R3` |
| I-тип | `OPCODE Rd, Rs1, #imm` | `ADDI R1, R2, #10` |
| J-тип | `OPCODE label` | `JMP loop` |
| O-тип | `OPCODE` | `HALT` |

#### Поддерживаемые инструкции

| Мнемоника | OPCODE (hex) | Тип | Операнды | Описание |
|-----------|--------------|-----|----------|----------|
| `NOP` | 0x00 | O | - | Нет операции |
| `HALT` | 0x01 | O | - | Остановка процессора |
| `NEG` | 0x02 | R | Rd, Rs1 | Арифметическое отрицание |
| `LOAD` | 0x10 | R | Rd, Rs1, [Rs2] | Загрузка из памяти по адресу Rs1+Rs2 |
| `LOADI` | 0x11 | I | Rd, #imm | Загрузка непосредственного значения |
| `STORE` | 0x12 | R | Rs1, [Rs2] | Сохранение в память по адресу Rs1+Rs2 |
| `ADD` | 0x20 | R | Rd, Rs1, Rs2 | Сложение |
| `SUB` | 0x21 | R | Rd, Rs1, Rs2 | Вычитание |
| `MUL` | 0x22 | R | Rd, Rs1, Rs2 | Умножение |
| `DIV` | 0x23 | R | Rd, Rs1, Rs2 | Деление |
| `ADDI` | 0x24 | I | Rd, Rs1, #imm | Сложение с константой |
| `SUBI` | 0x25 | I | Rd, Rs1, #imm | Вычитание константы |
| `AND` | 0x40 | R | Rd, Rs1, Rs2 | Побитовое И |
| `OR` | 0x41 | R | Rd, Rs1, Rs2 | Побитовое ИЛИ |
| `XOR` | 0x42 | R | Rd, Rs1, Rs2 | Побитовое исключающее ИЛИ |
| `MOV` | 0x30 | R | Rd, Rs1 | Копирование регистра |
| `CMP` | 0x31 | R | Rs1, Rs2 | Сравнение (устанавливает флаги Z, N, C) |
| `JMP` | 0x50 | J | label | Безусловный переход |
| `JZ` | 0x51 | J | label | Переход если Z=1 |
| `JNZ` | 0x52 | J | label | Переход если Z=0 |
| `JG` | 0x53 | J | label | Переход если больше (Z=0 и N=V) |
| `JL` | 0x54 | J | label | Переход если меньше (N!=V) |
| `JGE` | 0x55 | J | label | Переход если больше или равно (N=V) |
| `PUSH` | 0x70 | O | - | Поместить значение на стек |
| `POP` | 0x71 | O | - | Снять значение со стека |
| `CALL` | 0x72 | J | label | Вызов подпрограммы (PUSH PC, JMP) |
| `RET` | 0x73 | O | - | Возврат из подпрограммы (POP PC) |
| `PSTR` | 0x74 | R | Rs1 | Вывод Pascal-строки по адресу в Rs1 |
| `IN` | 0x75 | R | Rd | Ввод из порта (0xFF) |
| `OUT` | 0x76 | R | Rs1 | Вывод в порт (0xFE) |
